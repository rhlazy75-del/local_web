# ระบบถ่ายภาพ - Camera Capture System

## คำอธิบาย
ระบบนี้ประกอบด้วยสองส่วน:
1. **Flask Web Server** - เซิร์ฟเวอร์เว็บสำหรับการถ่ายภาพแบบเรียลไทม์
2. **Camera Capture** - ระบบดึงภาพจากกล้อง IP และบันทึกลงฐานข้อมูล

## ฟีเจอร์หลัก
✅ เปิด/ปิดกล้องผ่านหน้าเว็บ
✅ ถ่ายภาพแบบเรียลไทม์
✅ แสดงผลภาพที่ถ่าย
✅ ดาวน์โหลดภาพ
✅ ดึงภาพจากกล้อง IP อัตโนมัติ
✅ บันทึกลงฐานข้อมูล PostgreSQL

## การติดตั้ง

### 1. ติดตั้ง Dependencies
```bash
pip install -r requirements.txt
```

### 2. ตรวจสอบการตั้งค่า
แก้ไขค่าต่อไปนี้ใน `server.py`:
- `CAMERA_IPS` - IP address ของกล้อง
- `DB_CONFIG` - ข้อมูลการเชื่อมต่อ PostgreSQL
- `INTERVAL_SECONDS` - ช่วงเวลาการถ่ายภาพ

### 3. สร้างโฟลเดอร์ upload
```bash
mkdir upload_2cam
mkdir templates
```

โฟลเดอร์ `templates` ต้องมีไฟล์ `index.html` (ได้สร้างแล้ว)

## การใช้งาน

### เริ่มเซิร์ฟเวอร์
```bash
python server.py
```

### เข้าใช้เว็บ
เปิดเบราว์เซอร์ไปที่:
```
http://localhost:5000
```

## ข้อกำหนดเบราว์เซอร์
- Chrome / Edge / Firefox (เวอร์ชันล่าสุด)
- ต้องใช้ HTTPS หรือ localhost สำหรับการเข้าถึงกล้อง
- อนุญาตการเข้าถึงกล้องเมื่อเบราว์เซอร์ร้องขอ

## การแก้ไขปัญหา

### ไม่มีกล้องแสดง
- ตรวจสอบการอนุญาตในการตั้งค่าเบราว์เซอร์
- ตรวจสอบว่ากล้องไม่ถูกใช้โปรแกรมอื่น

### เชื่อมต่อฐานข้อมูลไม่ได้
- ตรวจสอบการตั้งค่า PostgreSQL ใน `server.py`
- เซิร์ฟเวอร์ยังคงทำงานแม้ไม่เชื่อมต่อ DB

### ดาวน์โหลดภาพไม่ได้
- ตรวจสอบตั้งค่าการดาวน์โหลดของเบราว์เซอร์
- ใช้เบราว์เซอร์ที่รองรับ Canvas API
